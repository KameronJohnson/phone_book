# phone_book
Week 2 Day 5 Epicodus assessment: Make a phone book that allows user to enter multiple phone numbers per contact

## Setup Instructions

###Gems used
```erb
sinatra
sinatra-contrib
```

###Install Bundler
```erb
$ gem install bundler
```

###Run Bundler
```erb
$ bundle
```
